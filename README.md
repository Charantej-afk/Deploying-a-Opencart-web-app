# AWS EC2 Apache Web Application Deployment

## Overview
This repository contains documentation and artifacts for a beginner-friendly project that demonstrates how to deploy a web application on **AWS EC2** using the **Apache2 web server**.

The project focuses on understanding core AWS concepts such as EC2 instances, security groups, SSH access, and hosting a web application that is accessible via a public IP address.

---

## Objective
- Launch an EC2 instance using the AWS Management Console  
- Configure security groups for SSH and HTTP access  
- Connect to the EC2 instance using SSH  
- Install and manage the Apache2 web server  
- Deploy and verify a sample web application
